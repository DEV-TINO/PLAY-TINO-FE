# PLAY-TINO-FE
Play-Tino Website FrontEnd Project by using `Vue.js`

# Skills
- Vue3.js
- Tailwind.css

# Installation

## Using docker
### Build docker image
```bash
docker compose build
```
### Run in background
```bash
docker compose up
```

### Show logs
```bash
docker compose logs -f
```

## Using node enviromnent
### Install Dependancy
```bash
npm install
```

### Run project for development
```bash
npm run dev
```

### Complie and Minify for Production
```bash
npm run build
```
